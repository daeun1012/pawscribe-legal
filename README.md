# pawscribe-legal

Pawscribe(AI 강아지 일기 앱)의 법적 안내 페이지를 호스팅하는 정적 사이트.

GitHub Pages로 배포되며, App Store / Google Play 심사에 필요한 URL을 제공한다.

## 페이지

| 파일 | 용도 | URL |
| --- | --- | --- |
| `index.html` | 랜딩 (법적 안내 인덱스) | `https://daeun1012.github.io/pawscribe-legal/` |
| `terms-of-service.html` | 이용약관 | `https://daeun1012.github.io/pawscribe-legal/terms-of-service.html` |
| `privacy-policy.html` | 개인정보처리방침 (PIPA + GDPR) | `https://daeun1012.github.io/pawscribe-legal/privacy-policy.html` |
| `account-deletion.html` | 계정 및 데이터 삭제 요청 안내 (Google Play 정책 필수) | `https://daeun1012.github.io/pawscribe-legal/account-deletion.html` |

## 수정

루트의 HTML 파일을 직접 편집한 뒤 `main`에 push하면 1~2분 내에 GitHub Pages가
자동 재배포한다.
